Projects and Experiences:

Drawing Robot

Project Overview

I built a robot that extracts line traces from an image and programed it to reproduce the images as a drawing on a whiteboard.

Components Used

  • Matlab/Simulink
  • Arduino Nano 33 IoT (microcontroller)
  • Arduino Nano Motor Carrier
  • Servo Motors
  • DC Motor Drivers
  • Frame and Structure
  • Markers
  • Power Supply

Skills

  • MATLAB programming and image processing
  • Microcontroller programming (Arduino)
  • Sensor integration
  • Mechanical design and assembly
  • Hardware troubleshooting and circuit analysis

Documentation

Day 0: Setting up connection between the microcontroller and MATLAB, as well as learning how to use MATLAB and SIMULINK through the arduino courseware.

Day 1: Assembling the frame and structure of the robot. Installed motors and motor drivers, wired the electronics.
